# Fitness Tracker App

A web-based fitness tracking application built with Python Flask that helps users monitor their workouts, nutrition, and progress toward fitness goals.

## Features

- **User Authentication**: Sign up and login system
- **Profile Setup**: Track height, weight, and fitness goals (weight loss, muscle building, maintenance)
- **Exercise Library**: Video demonstrations of gym exercises with ratings and comments
- **Workout Plans**: Personalized exercise routines based on user goals
- **Nutrition Guidance**: Daily meal recommendations and calorie tracking
- **Weekly Calendar**: Log exercises and food intake to monitor calorie balance
- **Progress Tracking**: Visualize calories gained/lost and fitness improvements

## Technology Stack

- **Backend**: Python Flask
- **Frontend**: HTML, CSS, JavaScript
- **Database**: (To be determined - SQLite/PostgreSQL)
- **Testing**: Pytest
- **Containerization**: Docker
- **CI/CD**: GitHub Actions
